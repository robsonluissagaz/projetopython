1. Para o programa funcionar completamente os arquivos credenciais.json, sagaztec.PNG e tellkinter.exe terão
que estar todos na mesma pasta.


2. O arquivo credenciais.json é usado para configurar seu email e senha para poder utilizar a função de chamados
do programa.


3. Vá até o seu gmail e adicione 3 tags 1. Chamados 2. Resolvidos 3. Lixeira. o programa vai mapear todos estas tags
e vai mostrar toda a informação em uma treeview. Você deverá configurar uma senha para apps no seu gmail, você encontra
esta opção nas configurações do seu gmail. Utilize o seu email e a senha de app no arquivo credenciais.json entre aspas
"".


4. Caso não queira utilizar a função de Chamados é só ignorar o erro inicial que aparecerá no programa pois ele tenta acessar o
servidor do gmail, mas como não foi configurado nenhum ele mostra uma mensagem de erro.


5. Na função de cadastro de equipamentos se atente para as áreas que indicam ** pois são obrigatórias para o cadastro dos
equipamentos, também recomendo inserir o usuário para poder acessar a função de log de enventos.


5. Para acessar a função de log de eventos você terá que cadastrar o equipamento com o respectivo nome de usuário que está registrado
no sistema operacional. Abra o CMD e digite hostname e aparecerá o hostname.


6. O arquivo tellkinter.py contém toda a estrutura de programação desenvolvida, útil para fazer algum tipo de engenharia reversa.
(fiquem a vontade).


7. Qualquer dúvida é só abrir uma discussão pelo Git que esterei atento para responder.
